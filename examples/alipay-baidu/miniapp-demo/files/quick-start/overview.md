# 快速开始概述

高德小程序是手机应用嵌入高德客户端的一种方法，有以下特点：
​
- 基于 Web 技术，学习成本低
- 一套代码同时支持 iOS 和 Android，接近原生体验
- 提供丰富的组件和 API（比如获取用户信息、本地存储、支付功能等）
​
开发者可在快速开始文档中详细了解高德小程序：
​
